# 数据文件

本仓库**不包含**原始数据（约 144 MB，超过 GitHub 单文件限制）。

## 获取方式

1. 打开 Kaggle：[Credit Card Fraud Detection](https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud)
2. 下载 `creditcard.csv`
3. 放到本目录，路径应为：

```
data/creditcard.csv
```

## 数据说明

- 284,807 笔交易，31 列（Time、Amount、V1–V28、Class）
- Class=1 为欺诈，Class=0 为正常；欺诈率约 0.172%
